开放式 GIS 客户端开发包

SuperMap iClient for JavaScript 9D 是空间信息和服务的可视化交互开发平台，是SuperMap服务器和在线GIS平台系列产品的统一JS客户端。支持多源数据地图，支持多终端，跨浏览器， 通过本产品可快速实现浏览器上美观、流畅的地图呈现。

快速入门

OpenLayers

引入 iclient9-openlayers.js 

<script src="../dist/iclient9-openlayers.js"></script>

将 iSevrer 发布的地图加载至 OpenLayers 的 map 中 

<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title>SuperMap REST 地图服务底图</title>
        <link href="//cdn.bootcss.com/openlayers/4.0.1/ol.css" rel="stylesheet">
        <script src="//cdn.bootcss.com/openlayers/4.0.1/ol-debug.js"></script>
        <script type="text/javascript" src="../../dist/iclient9-openlayers.js"></script>
    </head>
    <body>
        <div id="map"></div>
        <script type="text/javascript">
            var map = new ol.Map({
                target: 'map',
                view: new ol.View({
                    center: [100.94238281254322,35.496456056550095],
                    zoom: 4,
                    projection:"EPSG:4326"
                }),
                layers: [new ol.layer.Tile({
                    source: new ol.supermap.TileSuperMapRest({
                        url: "http://support.supermap.com:8090/iserver/services/map-world/rest/maps/World",
                        projection:"EPSG:4326"
                    }),
                })]
            });
        </script>
    </body>
</html>

Leaflet

引入 iclient9-leaflet.js 

<script type="text/javascript" src="../../dist/iclient9-leaflet.js"></script>

将 iSevrer 发布的地图加载至 Leaflet 的 map 中 

<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title>SuperMap REST 地图服务底图</title>
        <link rel="stylesheet" href="http://cdn.bootcss.com/leaflet/1.0.3/leaflet.css">
        <script type="text/javascript" src="http://cdn.bootcss.com/leaflet/1.0.3/leaflet.js"></script>
        <script type="text/javascript" src="../dist/iclient9-leaflet.js"></script>
    </head>
    <body>
        <div id="mapLeaflet"></div>
        <script type="text/javascript">
            var map = L.map('mapLeaflet', {
                center: L.CRS.EPSG3857.projection.unproject(L.point(12956184.324539, 4850540.9409086)),
                maxZoom: 18,
                zoom: 10
            });
            L.supermap.tiledMapLayer("http://support.supermap.com.cn:8090/iserver/services/map-china400/rest/maps/China").addTo(map);
        </script>
    </body>
</html>
